EU waste codes (EWC)
--------------------

This repository provides:

- the European Waste Codes (EWC) in machine-readable format(s)
- source document and instructions to regenerate the waste codes
- Python package to assist in (re)generating the codes and using them

Besides as files retrievable from this repository, the data is provided
as a data package (see http://frictionlessdata.io/data-packages) that
can be fetched from the Data Hub https://datahub.io).

All the resulting output files are in the data/ directory.

The Python code is in the ewc/ directory.
